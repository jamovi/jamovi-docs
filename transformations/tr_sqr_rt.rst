==========================
sqaure root transformation
==========================

-  | .
   | In the formula editor, you can use the ``SQR_ROOT()`` function to calculate square roots for a variable.
   | For example, to calculate square roots for the variable ``score``, enter:

   .. code-block:: text

      SQR_ROOT(score)
    | You can give the new variable a name, such as ``score_sqrt``
